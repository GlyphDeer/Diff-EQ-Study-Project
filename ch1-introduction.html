1.1 Background

A differential equation (DE) is any equation relating an unknown function to one or more of its derivatives.

The whole subject exists because models of physical systems naturally produce equations involving rates of change. Free fall gives $m\,\frac{d^2h}{dt^2} = -mg$; continuously compounded interest gives $\frac{dP}{dt} = \frac{r}{100}P$; an RLC circuit gives a second-order equation in the charge $q(t)$. In each case the "answer" we want is a function, not a number.

The three classifications

Every DE you meet gets sorted three ways. You should be able to do this at a glance.

ClassificationThe questionThe split
Type How many independent variables? ODE (ordinary) — one independent variable, ordinary derivatives.
PDE (partial) — two or more, partial derivatives.
Order What is the highest derivative present? The order of that highest derivative. $y'' + y = 0$ is second order.
Linearity Does the unknown appear only "linearly"? Linear or nonlinear (see test below).

Linearity test

An $n$-th order ODE is linear if it can be written

$$a_n(x)\,y^{(n)} + a_{n-1}(x)\,y^{(n-1)} + \dots + a_1(x)\,y' + a_0(x)\,y = g(x).$$

The defining features: the unknown $y$ and all its derivatives appear to the first power, are not multiplied together, and sit inside no nonlinear function (no $\sin y$, $e^{y}$, $\sqrt{y'}$, $y\,y'$, $y^2$). The coefficients $a_k$ and the forcing term $g$ may be any functions of the independent variable alone.

Watch for

Linearity is about the dependent variable, not the independent one. $y' + x^2 y = \cos x$ is linear — the $x^2$ and $\cos x$ are just coefficients. But $y' + y^2 = x$ is nonlinear because of the $y^2$, and $y'' + \sin(y) = 0$ (the pendulum) is nonlinear because $y$ sits inside $\sin$.

Notation worth keeping straight

  • Dependent variable — the unknown function (often $y$).
  • Independent variable — what it depends on (often $x$ or $t$).
  • Leibniz $\frac{dy}{dx}$ vs. prime $y'$ vs. dot $\dot y$ (time derivatives in mechanics) all mean the same thing; the book switches freely.

1.2 Solutions and Initial Value Problems

A solution is a function that, when substituted in, turns the equation into an identity that holds on some interval.

Explicit vs. implicit solutions

An explicit solution is given as $y = \phi(x)$ — the unknown is isolated. An implicit solution is a relation $G(x,y) = 0$ that defines one or more solutions without solving for $y$. For example, $x^2 + y^2 = 4$ implicitly defines solutions of $\frac{dy}{dx} = -\frac{x}{y}$; you verify it by differentiating the relation, not by isolating $y$.

How to verify a proposed solution

  1. Differentiate the candidate as many times as the order requires.
  2. Substitute the function and its derivatives into the DE.
  3. Simplify and check that both sides are equal for all $x$ in an interval.
  4. Note the interval of definition — where the function and its derivatives actually exist.

Families of solutions and the role of constants

Integrating introduces arbitrary constants, so a DE usually has infinitely many solutions. An $n$-th order equation generally carries an $n$-parameter family (the general solution). A particular solution is one member of that family with the constants pinned down.

Initial value problems (IVPs)

To pin the constants down you impose conditions. An initial value problem supplies the value of the unknown (and enough of its derivatives) at a single point:

$$\frac{dy}{dx} = f(x,y), \qquad y(x_0) = y_0.$$

A first-order IVP needs one condition; an $n$-th order IVP needs $n$ conditions, all stated at the same $x_0$. (Conditions given at different points are a boundary value problem, a different animal.)

Existence & uniqueness (first order)

For $\dfrac{dy}{dx} = f(x,y)$, $y(x_0)=y_0$: if $f$ and $\dfrac{\partial f}{\partial y}$ are both continuous on a rectangle around $(x_0, y_0)$, then a solution exists and is unique on some interval around $x_0$.

Two cautions this theorem teaches: (1) it is only a guarantee — failing the hypotheses does not prove there is no solution; (2) the interval of existence can be smaller than you expect, even when $f$ is nice everywhere.

Where uniqueness fails

$\dfrac{dy}{dx} = y^{1/3}$, $y(0)=0$ has more than one solution (including $y\equiv0$ and $y = (\tfrac{2}{3}x)^{3/2}$) because $\partial f/\partial y = \tfrac13 y^{-2/3}$ blows up at $y=0$. When a problem's initial point sits where $f$ or its $y$-derivative is undefined, expect trouble. This is exactly the issue flagged in Quiz 2, Problem 3.

Quick recall checklist

  • Classify any DE by type (ODE/PDE), order, and linearity.
  • State the linear form and apply the linearity test to the dependent variable only.
  • Verify explicit and implicit solutions by substitution / differentiation.
  • Distinguish general solution (with constants) from particular solution (constants fixed).
  • Set up a first-order IVP and know that one condition fixes one constant.
  • State the existence-and-uniqueness conditions and recognize when they fail.
