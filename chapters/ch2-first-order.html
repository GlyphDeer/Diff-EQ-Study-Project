How to pick a method

  1. Can you separate it as $\frac{dy}{dx} = g(x)h(y)$? → Separable (§2.2, below).
  2. Is it linear: $\frac{dy}{dx} + P(x)y = Q(x)$? → Integrating factor $\mu = e^{\int P\,dx}$ (§2.3).
  3. Written as $M\,dx + N\,dy = 0$ with $\partial M/\partial y = \partial N/\partial x$? → Exact (§2.4).
  4. Almost exact? → find a special integrating factor (§2.5).
  5. None of the above? → try a substitution: homogeneous, $G(ax+by)$, or Bernoulli (§2.6).

Most exam mistakes are misidentification, not algebra. Spend the first ten seconds naming the form.

2.1 Introduction: Motion of a Falling Body

This section motivates the whole chapter with one model. Newton's second law on a falling object subject to gravity and linear air resistance gives

$$m\frac{dv}{dt} = mg - bv, \qquad b > 0.$$

It is simultaneously separable and linear, so it previews both of the main techniques. The key qualitative idea introduced here is terminal velocity: as $t\to\infty$ the exponential transient dies out and $v \to mg/b$, the equilibrium where acceleration is zero. This exact model is Quiz 2, Problem 6.

2.2 Separable Equations

The first real technique: if the variables can be pulled to opposite sides, you just integrate.

A first-order equation is separable if it can be written so that all the $y$-dependence sits with $dy$ and all the $x$-dependence sits with $dx$:

$$\frac{dy}{dx} = g(x)\,h(y) \quad\Longrightarrow\quad \frac{dy}{h(y)} = g(x)\,dx.$$

Procedure

  1. Write $\dfrac{dy}{dx}$ as a product of a function of $x$ alone and a function of $y$ alone. If you cannot, it is not separable.
  2. Divide so each variable is with its own differential: $\dfrac{1}{h(y)}\,dy = g(x)\,dx$.
  3. Integrate both sides. One constant of integration is enough (combine them on one side).
  4. Solve for $y$ if possible (explicit), or leave the relation as an implicit solution.
  5. Apply the initial condition to fix the constant.

Watch for

  • Lost solutions. Dividing by $h(y)$ assumes $h(y)\neq0$. Any constant $y=r$ with $h(r)=0$ is also a solution (an equilibrium) and must be checked separately — it can be missed entirely by the integration.
  • A sum like $\dfrac{dy}{dx} = x + y$ is not separable; you cannot factor it into $g(x)h(y)$. (It is linear, though — use §2.3.)
  • Keep the absolute value when integrating $\int \frac{dy}{y} = \ln|y|$, then resolve the sign with the initial condition.

Pattern: exponential growth/decay

$\dfrac{dy}{dx} = ky$ separates to $\dfrac{dy}{y} = k\,dx$, giving $\ln|y| = kx + C$ and $y = Ae^{kx}$. This is the backbone of the interest, radioactive-decay, and Newton's-law-of-cooling models. The falling-body equation of §2.1 is separable by exactly this route.

2.3 Linear Equations

The single most important first-order technique. Learn it cold.

A first-order linear equation can always be put in standard form:

$$\frac{dy}{dx} + P(x)\,y = Q(x).$$

Integrating-factor procedure

  1. Put the equation in standard form (coefficient of $y'$ must be 1). Identify $P(x)$ and $Q(x)$.
  2. Compute the integrating factor $\;\mu(x) = e^{\int P(x)\,dx}$. (No constant needed — see note.)
  3. Multiply through by $\mu$. The left side collapses to a product derivative: $\dfrac{d}{dx}\big[\mu(x)\,y\big] = \mu(x)\,Q(x)$.
  4. Integrate both sides: $\mu(x)\,y = \displaystyle\int \mu(x)\,Q(x)\,dx + C$.
  5. Solve for $y$ and apply any initial condition to find $C$.

Why it works: $\mu$ is built precisely so that $\mu' = \mu P$. Then the product rule gives $(\mu y)' = \mu y' + \mu' y = \mu y' + \mu P y = \mu(y' + Py) = \mu Q$, which is the whole point — the left side becomes an exact derivative you can integrate.

Common errors

  • Forgetting to divide so the leading coefficient is 1 before reading off $P$.
  • Multiplying only one side by $\mu$.
  • Dropping the constant of integration, or adding it on the wrong side.
  • The constant inside $\mu = e^{\int P\,dx + c}$ is harmless — $e^c$ factors out and cancels. Conventionally take it to be zero. (Quiz 2, Problem 4 is exactly this point.)

Pattern: when $Q$ has no elementary antiderivative

For $y' - 2xy = 1$, the factor is $\mu = e^{-x^2}$ and you reach $e^{-x^2}y = \int e^{-x^2}dx + C$. That integral is non-elementary, so the answer is written with the error function $\operatorname{erf}(x) = \frac{2}{\sqrt\pi}\int_0^x e^{-t^2}dt$. Don't panic when this happens — leaving the answer in terms of $\operatorname{erf}$ (or a definite integral) is the correct, complete answer. (Quiz 2, Problem 9.)

2.4 Exact Equations

Write the equation in differential form:

$$M(x,y)\,dx + N(x,y)\,dy = 0.$$

It is exact when its left side is the total differential $dF$ of some potential function $F(x,y)$. The solutions are then the level curves $F(x,y) = C$.

Test and solve

  1. Test: the equation is exact $\iff \dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}$ (on a simply connected region).
  2. Integrate $M$ with respect to $x$: $F = \displaystyle\int M\,dx + g(y)$, where $g(y)$ is an unknown "constant" of integration that may depend on $y$.
  3. Differentiate that $F$ with respect to $y$, set it equal to $N$, and solve for $g'(y)$.
  4. Integrate $g'(y)$ to get $g(y)$. Assemble $F$.
  5. The implicit solution is $F(x,y) = C$.

(You can equally start by integrating $N$ with respect to $y$; pick whichever is easier.)

Watch for

Always run the equality-of-mixed-partials test before committing. If $M_y \neq N_x$ the method does not apply as-is — you need §2.5. And the function added during the first integration depends on the other variable (a $g(y)$, not a bare constant); that is where the new information comes from.

2.5 Special Integrating Factors

When $M\,dx + N\,dy = 0$ fails the exactness test, you may be able to multiply by a factor $\mu$ that makes it exact. In general $\mu$ depends on both variables and is hard to find, but two special cases are tractable and worth memorizing.

If this quantity depends on……only this variableIntegrating factor
$\dfrac{M_y - N_x}{N}$ $x$ alone $\mu(x) = \exp\!\left(\displaystyle\int \frac{M_y - N_x}{N}\,dx\right)$
$\dfrac{N_x - M_y}{M}$ $y$ alone $\mu(y) = \exp\!\left(\displaystyle\int \frac{N_x - M_y}{M}\,dy\right)$

Procedure

  1. Confirm the equation is not already exact.
  2. Compute $\dfrac{M_y - N_x}{N}$. If it is a function of $x$ only, use the $x$-factor.
  3. Otherwise compute $\dfrac{N_x - M_y}{M}$. If it is a function of $y$ only, use the $y$-factor.
  4. Multiply the whole equation by $\mu$ and solve the now-exact equation by §2.4.

Note the sign flip and the denominator swap ($N$ for the $x$-case, $M$ for the $y$-case). Mixing these up is the usual error.

This section also re-derives the §2.3 linear integrating factor as a special case: a linear equation in differential form has $\dfrac{M_y - N_x}{N} = P(x)$, recovering $\mu = e^{\int P\,dx}$.

2.6 Substitutions and Transformations

When none of the standard forms fit, a change of variable can convert the equation into one that does. Three named substitutions cover most cases.

1 · Homogeneous equations

If $\dfrac{dy}{dx} = f(x,y)$ where $f$ depends only on the ratio $y/x$ (equivalently $M,N$ are homogeneous of the same degree), substitute

$$v = \frac{y}{x}, \qquad y = vx, \qquad \frac{dy}{dx} = v + x\frac{dv}{dx}.$$

The result is always separable in $v$ and $x$. Solve, then back-substitute $v = y/x$.

2 · Equations of the form $\frac{dy}{dx} = G(ax + by)$

Substitute $z = ax + by$. Then $\dfrac{dz}{dx} = a + b\dfrac{dy}{dx} = a + bG(z)$, which is separable in $z$ and $x$.

3 · Bernoulli equations

An equation of the form

$$\frac{dy}{dx} + P(x)\,y = Q(x)\,y^{n}, \qquad n \neq 0, 1$$

becomes linear under the substitution $v = y^{1-n}$. Then $\dfrac{dv}{dx} = (1-n)y^{-n}\dfrac{dy}{dx}$, and dividing the original by $y^n$ and substituting gives

$$\frac{dv}{dx} + (1-n)P(x)\,v = (1-n)Q(x),$$

which you solve by the §2.3 integrating factor, then back-substitute $v = y^{1-n}$.

Watch for

  • $n = 0$ or $n = 1$ are not Bernoulli — those are already linear or separable. The substitution is only needed for other powers.
  • For homogeneous equations, confirm $f$ truly reduces to a function of $y/x$ before substituting; a stray additive $x$ or constant breaks it (try the $G(ax+by)$ form instead).
  • Always back-substitute to return to the original variable. A common slip is leaving the answer in $v$.

Quick recall checklist

  • Name the form first: separable, linear, exact, almost-exact, or substitution-needed.
  • Linear: standard form → $\mu = e^{\int P\,dx}$ → $(\mu y)' = \mu Q$ → integrate.
  • Exact: test $M_y = N_x$ → integrate $M$ in $x$ → fix $g(y)$ via $N$.
  • Special factor: check $\frac{M_y - N_x}{N}$ ($x$-only) or $\frac{N_x - M_y}{M}$ ($y$-only).
  • Substitutions: $v=y/x$ (homogeneous), $z=ax+by$, $v=y^{1-n}$ (Bernoulli).
  • Always back-substitute and apply initial conditions last.
