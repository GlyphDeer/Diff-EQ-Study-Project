Quiz 1: Classification, Solutions & IVPs

Prerequisite Topics to Review

  • Calculus Rules: Mastery of product, quotient, and chain rules. Familiarity with partial differentiation.
  • Algebraic Manipulation: Factoring polynomials, exponent rules, and solving characteristic equations.
  • Domain & Continuity: Identifying domain restrictions (e.g., division by zero, negative square roots) in two-variable functions.

Method Recaps

Classification: Determining the algebraic structure of the differential equation (DE) to select the correct solving tool.

Solution Verification: Proving a given function satisfies a DE by computing its derivatives and substituting them into the original equation to verify it creates a true identity for all $x$.

Existence and Uniqueness (E&U): Determining if an Initial Value Problem (IVP) is guaranteed to have exactly one solution without actually solving it, relying on the continuity of the right-hand side function and its partial derivative.

Framework 1: Differential Equation Classification

Use this ordered checklist to classify any DE.

  1. Type: Count the independent variables (what you differentiate with respect to).
    → 1 variable = ODE.
    → 2+ variables = PDE.
  2. Order: Locate the highest derivative present in the entire equation.
    → The number of that highest derivative is the Order.
  3. Linearity: Check the dependent variable (usually $y$) and all of its derivatives.
    → If $y$ and its derivatives are to the 1st power, are not multiplied together, and do not sit inside other functions (like $\sin(y)$ or $e^y$), it is Linear. Otherwise, it is Nonlinear.

Framework 2: Existence & Uniqueness Test

Use this algorithm to evaluate IVPs of the form $y' = f(x,y), \quad y(x_0) = y_0$.

  1. Isolate: Rewrite the ODE to isolate the first derivative: $y' = f(x,y)$.
  2. Test $f(x,y)$: Determine if $f(x,y)$ is continuous at and immediately around the initial point $(x_0, y_0)$. Check for zero denominators or invalid function arguments.
  3. Differentiate: Compute the partial derivative with respect to the dependent variable: $\frac{\partial f}{\partial y}$.
  4. Test $\frac{\partial f}{\partial y}$: Determine if the partial derivative is continuous at and around $(x_0, y_0)$.
  5. Conclude: If both are continuous near $(x_0, y_0)$, a unique solution exists. If either fails, uniqueness (or existence) is not mathematically guaranteed.

Quiz 2: First-Order ODEs

Prerequisite Topics to Review

  • Integration Techniques: Integration by parts, $u$-substitution, integration of rational functions, and recognizing non-elementary integrals (like $\int e^{-x^2}dx$).
  • Exponential & Logarithmic Properties: Manipulating base $e$ and natural logarithms ($\ln$) to isolate variables.
  • Physics/Engineering Translation: Translating physical laws (Newton's Second Law, Newton's Law of Cooling, mass/brine balance) into rate-of-change differential equations.

Method Recaps

Separation of Variables: Isolating all $y$ terms with $dy$ on one side and $x$ terms with $dx$ on the other, allowing direct integration of both sides.

Integrating Factors: Multiplying a non-separable linear equation by a specially computed factor $\mu(x)$ to force the left side to become the exact derivative of a product, making it integrable.

Applied Modeling: Setting up an IVP by identifying the rate of change ($\frac{dy}{dt}$), the forcing functions (inflow/outflow, ambient temperature), and the initial state ($y(0)$).

Framework 3: First-Order Method Selection

Apply this hierarchy to determine the most efficient solving method.

  1. Separable? Can it be factored into $y' = g(x)h(y)$?
    → Yes: Separate and integrate: $\int \frac{dy}{h(y)} = \int g(x)dx$.
  2. Linear? Can it be written as $y' + P(x)y = Q(x)$?
    → Yes: Use the Integrating Factor Framework (Framework 4).
  3. Exact? Written as $M\,dx + N\,dy = 0$. Does $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$?
    → Yes: Integrate $M$ with respect to $x$, then find $g(y)$ using $N$.

Framework 4: Integrating Factor Algorithm

For linear equations that are not strictly separable.

  1. Standardize: Divide the equation by the leading coefficient to achieve standard form: $y' + P(x)y = Q(x)$.
  2. Compute $\mu(x)$: Calculate the integrating factor: $\mu(x) = e^{\int P(x)dx}$. (You can set the constant of integration to zero here).
  3. Multiply: Multiply the entire standard form DE by $\mu(x)$. The left side automatically collapses to the product rule: $(\mu(x)y)' = \mu(x)Q(x)$.
  4. Integrate: Integrate both sides with respect to $x$: $\mu(x)y = \int \mu(x)Q(x)dx + C$. (Ensure $+ C$ is explicitly added right now!)
  5. Solve: Divide by $\mu(x)$ to isolate $y$. Apply initial conditions to solve for $C$.

Framework 5: Mixture/Compartment Model Setup

For fluid mass-balance problems (e.g., salt in a tank). Ensure all units match before starting.

  1. Variables: Let $A(t)$ be the mass of the solute. Let $V(t)$ be the volume of the fluid.
  2. Rate In: Calculate input rate = (Concentration In) $\times$ (Flow Rate In).
  3. Rate Out: Calculate output rate = $\left(\frac{A(t)}{V(t)}\right) \times$ (Flow Rate Out).
    Note: $V(t) = V_0 + (\text{Flow In} - \text{Flow Out})t$.
  4. Construct DE: Form the equation: $\frac{dA}{dt} = \text{Rate In} - \text{Rate Out}$.
  5. Solve: This setup almost always results in a linear 1st-order DE. Use Framework 4 to solve for $A(t)$.
